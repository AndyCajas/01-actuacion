Introducción

Flexbox y CSS Grid son dos sistemas de layout que han revolucionado la forma en que creamos diseños web. Mientras que Flexbox es ideal para diseños unidimensionales (en una fila o columna), CSS Grid es perfecto para layouts bidimensionales complejos.

Flexbox

Sistema unidimensional para diseñar layouts en una sola dimensión (filas o columnas). Perfecto para componentes y diseños pequeños.

CSS Grid

Sistema bidimensional que permite controlar tanto filas como columnas simultáneamente. Ideal para layouts complejos y estructuras generales.

Flexbox

Conceptos Clave

Flexbox se basa en un modelo de contenedor y elementos hijos (flex items). El contenedor flex define cómo se distribuyen los elementos dentro de él.

Ejemplo Visual:

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    gap: 1rem;
}

Propiedades Principales

Propiedad Descripción Valores comunes
display: flex Define un contenedor flex flex | inline-flex
flex-direction Dirección de los elementos row | row-reverse | column | column-reverse
justify-content Alineación en el eje principal flex-start | flex-end | center | space-between | space-around | space-evenly
align-items Alineación en el eje cruzado flex-start | flex-end | center | baseline | stretch
flex-wrap Permite el ajuste de línea nowrap | wrap | wrap-reverse
gap Espacio entre elementos Valores en px, rem, em, etc.

Casos de Uso

Menús de navegación

Perfecto para crear menús horizontales o verticales con distribución uniforme.

Centrado vertical

La forma más sencilla de centrar elementos vertical y horizontalmente.

Controles de formulario

Alinear etiquetas y campos de formulario de manera consistente.

Tarjetas de productos

Crear filas de elementos con altura uniforme independientemente del contenido.

CSS Grid

Conceptos Clave

CSS Grid permite crear layouts complejos dividiendo el espacio en filas y columnas. Se basa en un sistema de rejilla con áreas nombradas.

Ejemplo Visual:

Header
Sidebar
Main
Footer
.container {
    display: grid;
    grid-template-columns: 1fr 2fr;
    grid-template-rows: auto 1fr auto;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
    gap: 1rem;
}

Propiedades Principales

Propiedad Descripción Valores comunes
display: grid Define un contenedor grid grid | inline-grid
grid-template-columns Define las columnas de la rejilla fr, px, %, auto, minmax(), repeat()
grid-template-rows Define las filas de la rejilla fr, px, %, auto, minmax(), repeat()
grid-template-areas Asigna nombres a áreas de la rejilla Nombres de áreas
gap Espacio entre celdas Valores en px, rem, em, etc.
grid-column/grid-row Posicionamiento de elementos hijos Números de línea, span, nombres de área

Casos de Uso

Layouts de página completa

Estructuras complejas con header, footer, sidebar y contenido principal.

Galerías de imágenes

Crear mosaicos de imágenes con diferentes tamaños y proporciones.

Dashboard

Organizar widgets y componentes en una cuadrícula flexible.

Diseños asimétricos

Crear layouts artísticos que rompan con la estructura tradicional.

¿Cuándo usar cada uno?

Usa Flexbox cuando...

  • Necesitas distribuir elementos en una sola dimensión (fila o columna)
  • Tienes un conjunto lineal de elementos que deben alinearse de cierta manera
  • Quieres controlar el espacio entre elementos de manera uniforme
  • Necesitas centrar elementos vertical u horizontalmente
  • Trabajas con componentes pequeños como menús, barras de herramientas o grupos de botones

Usa CSS Grid cuando...

  • Necesitas controlar el layout en dos dimensiones (filas y columnas)
  • Quieres crear estructuras complejas con áreas nombradas
  • Necesitas superponer elementos o crear layouts asimétricos
  • Trabajas con el layout general de la página
  • Quieres crear diseños responsivos sin necesidad de media queries

¿Puedo usarlos juntos?

¡Absolutamente! Flexbox y CSS Grid no son excluyentes. De hecho, se complementan perfectamente. Puedes usar:

  • CSS Grid para el layout general de la página
  • Flexbox para los componentes dentro de cada área del grid

Esta combinación ofrece el máximo poder y flexibilidad para crear diseños modernos y responsivos.

Ejemplos Prácticos

Ejemplo 1: Layout de página con Grid + componentes con Flexbox

<div class="page-layout">
    <header>Header</header>
    <nav>
        <ul class="nav-menu">
            <li>Inicio</li>
            <li>Productos</li>
            <li>Contacto</li>
        </ul>
    </nav>
    <main>Contenido principal</main>
    <aside>Sidebar</aside>
    <footer>Footer</footer>
</div>

<style>
    .page-layout {
        display: grid;
        grid-template-columns: 1fr 3fr;
        grid-template-rows: auto 1fr auto;
        grid-template-areas:
            "header header"
            "nav main"
            "footer footer";
        min-height: 100vh;
    }
    
    .nav-menu {
        display: flex;
        flex-direction: column;
        gap: 1rem;
    }
</style>

Ejemplo 2: Galería responsiva

<div class="gallery">
    <div class="gallery-item"></div>
    <div class="gallery-item"></div>
    <div class="gallery-item"></div>
    <!-- más items -->
</div>

<style>
    .gallery {
        display: grid;
        grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
        gap: 1rem;
    }
    
    @media (max-width: 600px) {
        .gallery {
            grid-template-columns: 1fr 1fr;
        }
    }
</style>
